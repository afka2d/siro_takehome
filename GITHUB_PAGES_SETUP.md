# GitHub Pages Setup Instructions

## Enable GitHub Pages

1. Go to your repository: https://github.com/afka2d/siro_takehome
2. Click on **Settings** (top menu)
3. Scroll down to **Pages** (left sidebar)
4. Under **Source**, select:
   - Branch: `main`
   - Folder: `/ (root)`
5. Click **Save**

## Access Your Presentation

Once GitHub Pages is enabled (may take 1-2 minutes), your presentation will be available at:

**https://afka2d.github.io/siro_takehome/**

## Presentation Features

- **Navigation:** Use arrow keys or click navigation buttons
- **Fullscreen:** Press `F` key
- **Overview:** Press `ESC` key to see all slides
- **Code Links:** Click any "View Code" link to see the exact code section

## Files Included

- `index.html` - Web-based reveal.js presentation
- `siro_sales_analysis_20251120044535.pptx` - Original PowerPoint file (for reference)
- `.nojekyll` - Prevents Jekyll processing

## Troubleshooting

If the page doesn't load:
1. Wait 1-2 minutes after enabling GitHub Pages
2. Check that the branch is set to `main` and folder is `/ (root)`
3. Clear your browser cache
4. Try accessing: `https://afka2d.github.io/siro_takehome/index.html`
